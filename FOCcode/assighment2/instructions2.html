Problem 1

Solve this problem inside the file problem1.c.

In this problem you will write a program to calculate whether a number is a "Ronaldo Number" or not. A number is considered a Ronaldo Number if the sum of its digits, each to the power of the number of its digits, is either:

  • Equal to the number
  • Equal to the number minus 7
  • Equal to the number plus 7
  • Equal to the number times 7
  • Equal to the number divided by 7 (integer division)

For example, the number 407 is a Ronaldo Number. It has three digits 4, 0 and 7, and the number of its digits is 3. If we raise each digit to 3 and sum them together we get: 4^3 + 0^3 + 7^3 = 64 + 0 + 343 = 407 which is equal to the number itself.

92 is also a Ronaldo Number. It has 2 digits 9 and 2, and the number of its digits is 2. If we raise each digit to 2 and sum them together we get: 9^2 + 2^2 = 81 + 4 = 85 which is equal to the number itself minus 7.

Accept the input WITHOUT printing any sentences like "Please enter a number". If you print anything extra it will be considered part of the answer and will make your answer wrong.

Input Output

Two friends called Leena and Samar are participating in a contest against each other. The contest has r rounds, and each round has a different number of challenges. Here is how winners are decided:

  • The winner of a round is the friend that wins more than half the challenges in that round. If a friend wins more than half the challenges at any time during the round, the round immediately ends.

Problem 7

Solve this problem inside the file problem7.c. Do not modify any other files for this problem.

In this problem you will write a program that finds the Greatest Common Divisor and the Least Common Multiple between n integers.

The Greatest Common Divisor (GCD) between two integers is the largest positive integer that the 2 integers can be divided by without a remainder. For example, the GCD of 12 and 20 is 4, because 4 is the largest integer both 12 and 20 are divisible by.

The Least Common Multiple (LCM) between two integers is the smallest positive integer that is divisible by the 2 integers without a remainder. For example, the LCM of 12 and 20 is 60, because 60 is the smallest integer that is divisible by both 12 and 20.

Getting Started

  • Your program must accept n, the number of integers. Then it must accept n integers, and finally print the Greatest Common Divisor between all the n integers and the Least Common Multiple between all the n integers.
  • You are given a function called "greatestCommonDivisor" that accepts 2 integers and returns the GCD of them. You can read about the function and how to use it by going to the file problem7-library.h. You must use this function to find the GCD of all n integers. If you are interested in seeing how it works on the inside, go to the file problem7-library.c.
  • To find the LCM of two integers, you can simply multiply the two integers then divide them by their GCD. For example, LCM(12, 20) = (12 x 20) divided by GCD(12, 20) = 240 divided by 4 = 60.
  • We can use the function "greatestCommonDivisor" and the LCM formula from above to find the GCD and the LCM of 2 integers only, but we need to find the GCD and the LCM between all n integers. Luckily for us, GCD and LCM are associative, which means that they can be applied to more than just 2 integers by simply applying them to the integers in pairs.

Example: To find the GCD of 12, 20, and 34 we simply find the GCD of 12 and 20 first, which is 4, then we find the GCD of 4 and 34, which is 2. The final answer 2 is the GCD of the 3 numbers together.

Example: To find the LCM of 12, 20, and 34 we simply find the LCM of 12 and 20 first, which is 60, then we find the LCM of 60 and 34, which is 1020. The final answer 1020 is the LCM of the 3 numbers together.

Accept the input WITHOUT printing any sentences like "Please enter a number". If you print anything extra it will be considered part of the answer and will make your answer wrong.
